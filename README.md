## 📊 Dataset Overview

The dataset contains **5,901 records** with **23 columns**:

| Column | Description |
|--------|-------------|
| Order ID | Unique order identifier |
| Order Date / Ship Date | Order and shipping dates |
| Ship Mode | Shipping method used |
| Customer ID / Name | Customer details |
| Segment | Customer segment (Consumer, Corporate, Home Office) |
| Country / City / State / Region | Delivery location |
| Category / Sub-Category | Product classification |
| Product Name | Name of the product |
| Sales | Revenue from order |
| Quantity | Units ordered |
| Profit | Profit earned |
| Returns | Return status |
| Payment Mode | Online, Cards, COD |

---

## 📈 Dashboard Highlights

- 📅 **Sales & Profit by Month** — Monthly trend analysis with YoY comparison
- 🗺️ **Sales by State** — Geographic map of sales distribution across the US
- 🏷️ **Sales by Category & Sub-Category** — Top performing product categories
- 🚚 **Sales by Ship Mode** — Delivery method performance breakdown
- 👥 **Sales by Segment** — Consumer vs Corporate vs Home Office
- 💳 **Sales by Payment Mode** — Online vs Cards vs COD split
- 📦 **Sales by Region** — East, West, Central, South comparison
- 🔮 **15-Day Sales Forecast** — Time series forecasting using Power BI built-in analytics

---

## 🔍 Key Insights

- 🏆 **Consumer** segment contributes the highest share of sales
- 📦 **Standard Class** is the most preferred shipping mode
- 🗓️ Sales peak during **Q4** (October–December)
- 🖥️ **Technology** is the highest revenue-generating category
- 💰 **COD (Cash on Delivery)** is the most used payment method
- 📍 **California** leads all states in total sales

## 🛠️ Tools & Features Used

- **Power BI Desktop**
- Data Cleaning & Transformation (Power Query)
- DAX Measures & Calculated Columns
- Interactive Slicers (Region filter)
- Map Visualization
- Line Chart with Forecasting (15-day prediction)
- Donut Charts, Bar Charts, Area Charts
- KPI Cards (Total Sales, Profit, Quantity, Avg Delivery Days)

---

## 🚀 How to Use

1. Download `Dashboard_2_superstore.pbix`
2. Open it in **Power BI Desktop** (free download from Microsoft)
3. Explore the dashboard using the **Region slicer** to filter by East / West / Central / South
4. View the **Sales Forecast** section for the 15-day prediction
